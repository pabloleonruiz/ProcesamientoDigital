NAME

     watershed_segment - image segmentation based on the watershed segmentation
                         algorithm
 

SYNOPSIS

      #include "watershed.h"
      #include "CVIPmap.h"
 
     Image *watershed_segment(Image *inputImage, float threshold, CVIP_BOOLEAN choice)
 
      <inputImage> - pointer to Image structure
      <threshold> - threshold value
      <choice> - CVIP_NO (0) Do not merge result
                 CVIP_YES (1) To merge result

PATH

     $CVIPtoolsHOME\CVIPC\SEGMENTATION\watershed_segment.c
 

DESCRIPTION

     The watershed segmentation algorithm as implemented in CVIPtools was initially designed to separate a single object from the background in color images. The watershed algorithm is a morphological technique based on the idea of modeling a gray level image as a topographic surface, with higher gray levels corresponding to higher elevations. The image is then flooded with a rainfall simulation, and pools of water are created corresponding to segments within the image. When rising water reaches a point where two pools will merge, a dam is built to prevent the merging.

RETURN VALUES

     Pointer to an Image structure on success and NULL pointer
     on failure.
