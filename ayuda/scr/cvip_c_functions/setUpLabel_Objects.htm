NAME

     setUpLabel_Objects - calls object labeling
 

SYNOPSIS

      #include "CVIPtoolkit.h"
      #include "CVIPobject.h"
 
     Image * setUpLabel_Objects( const Image *imageP )
 
      <imageP> - pointer to an Image
 

PATH

     $CVIPtoolsHOME\CVIPC\Object\ObjectSetUp.c

DESCRIPTION

     This function calls the function label_Objects.
 

TYPES AND CONSTANTS

     None
 

RETURN VALUES

     A labeled image
 

HISTORY

     History information recorded: None
