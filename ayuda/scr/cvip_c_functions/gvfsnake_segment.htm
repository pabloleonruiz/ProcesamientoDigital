NAME

     gvfsnake_segment - an active contour segmentation technique. 
 

SYNOPSIS

      #include "CVIPtoolkit.h"
      #include "CVIPdef.h"
      #include "CVIPsegment.h"
 
      Image *gvfsnake_segment(Image *inputImage, int num_iter, double alpha, double beta, double kappa)
 
      <inputImage>     - pointer to an Image structure
        <num_iter>     - number of iterations
           <alpha>     - coefficient of the second derivative of the function in the energy equation
            <beta>     - coefficient of the fourth derivative of the function in the energy equation
           <kappa>     - coefficient of the external forces in the energy equation 
 
     

PATH

     $CVIPtoolsHOME\CVIPC\SEGMENTATION\gvfsnake.c
 

DESCRIPTION

The gvf snake method can be used for edge detection,

segmentation, shape modeling and motion tracking.

A snake, is a curve defined within an image domain that

can move under the influence of internal forces within

the curve itself and external forces derived from the

image data. The internal and external forces are defined

so that the snake will conform to an object boundary

or other desired features within an image. This function
calculates the external forces by calculating the gradient 
vector flow fields in the image.
The method works only on gray level images. Though the
function itself converts the image even if the input
image is a color image. 
     

TYPES AND CONSTANTS

        None.

    

RETURN VALUES

     Returns an Image pointer upon successful completion, and
     returns NULL on failure.
 

EXAMPLE

      #include "CVIPtoolkit.h"
      #include "CVIPdef.h"
      #include "CVIPconvert.h"
      #include "CVIPsegment.h"
      #include "CVIPimage.h"
 
           void main()
           {
               Image *cvipImage;
               int num_iter;
                double alpha, beta, kappa;
                char *inputfile;
 

                             print_CVIP("\n\t\tEnter the Input File Name:  ");

              inputfile = getString_CVIP();

              cvipImage = read_Image( inputfile ,1);

 
               print_CVIP("\n\t\tEnter the number of itearations:  ");
               num_iter = getInt_CVIP(1, 0, 1000);
           
                print_CVIP("\n\t\tEnter the alpha coefficient:  ");
                alpha = getUInt_CVIP(0.01, 0, 0.2);
           
               print_CVIP("\n\t\tEnter the beta coefficient:  ");
                beta = getUInt_CVIP(0.01, 0, 0.2);
               
                print_CVIP("\n\t\tEnter the kappa coefficient:  ");
                kappa = getUInt_CVIP(0.01, 0, 0.2);
           
                cvipImage = gvfsnake_segment(cvipImage, num_iter, alpha, beta, kappa);
 
               view_Image(cvipImage, "GVF Snake");
 
               free(inputfile);
                delete_Image (cvipImage);
           }
