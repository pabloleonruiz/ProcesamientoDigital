NAME

     marr_hildreth_filter – performs a Marr Hildreth edge detection on the image
 

SYNOPSIS

      #include <CVIPtoolkit.h>
      #include <CVIPconvert.h>
      #include <CVIPdef.h>
      #include <CVIPimage.h>
      #include <CVIPspfltr.h>
 
     Image * marr_hildreth_filter (Image* inputImage, float sigma)
 
      <inputImage> - pointer to the input Image structure
      <sigma> - Sigma value
 

PATH

     $CVIPtoolsHOME\CVIPC\SPATIALFILTER\ marr_hildreth_filter.c
 

DESCRIPTION

Marr-Hildreth Filter works by performing three steps:

1. Convolve the image with a Gaussian smoothing filter

2. Convolve the image with a Laplacian mask

3. Find the zero crossings of the image from Step 2.

TYPES AND CONSTANTS

     None
 

RETURN VALUES

     Returns the edge detected image
