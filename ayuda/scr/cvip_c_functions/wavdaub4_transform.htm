NAME

     wavdaub4_transform - performs a Daubechies wavelet transform
 

SYNOPSIS

      #include <stddef.h>
      #include <ctype.h>
      #include <string.h>
      #include <math.h>
      #include "CVIPimage.h"
      #include "CVIPtools.h"
      #include "CVIPdef.h"
      #include "wavelet.h"
 
     Image  *wavdaub4_transform(Image  *image,  int  isign,   int
     lowband)
 
      <image> - pointer to an Image structure
      <isign> - 1 (forward transform) or 2 (inverse transform)
      <lowband> - # of rows/(2^([(# bands desired -1)/3]-1))
 

PATH

     $CVIPtoolsHOME\CVIPC\TRANSFORMS\wavelet.c

DESCRIPTION

     This function performs a Daubechies 4 tap wavelet filter  in
     the spatial domain.
 

TYPES AND CONSTANTS

     None
 

RETURN VALUES

     A transformed image
 

HISTORY

     History information recorded:
 
     ** Wavelet Transform:
          - Basis: <type> wavelet
          - # of frequency bands: <#>
