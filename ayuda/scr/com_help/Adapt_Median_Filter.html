Description

 
Ranked-order based adaptive median filter is used to  remove
     positive and negative impulse noises in an image.
      The Algorithm:
      Xmed = median value in window
      Xmin = minimum value in window
      Xmax = maximum value in window
      Xrc = current pixel value at (r,c)
      (Level 1) T1 = Xmed - Xmin, T2 = Xmed - Xmax
      (Level 2) U1 = Xrc - Xmin,  U2 = Xrc - Xmax
      Level 1: If T1>0&T2<0, goto Level 2
               Else increase window size
      Level 2: If U1>0&U2<0, output = Xrc
               Else output=Xmed
 
     For more details see H. Hwang, R.A. Haddad, "Adaptive Median
     Filters: New Algorithms and Results", IEEE Trans. Image Pro-
     cessing, Vol. 4, pp. 499-501, April 1995.
