Description

This function applies prewitt edge detection algorithms to <imageP>.

 If a smoothing filter is desired as a preprocessing step, set <mask_choice> to:

 1 = Gaussian blur

 2 = generic lowpass 1

 3 = generic lowpass 2

 4 = neighborhood average

 For <mask_size>: set mask size to any desired kernel size

 

NOTE: FOR ALL OF THE ABOVE PARAMETERS, USE THE VALUE -1 FOR ANY PARAMETER NOT NEEDED.
