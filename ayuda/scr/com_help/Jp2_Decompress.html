Name

Jpg_Decompress

Joint Photographic Experts Group(JPEG) image compression scheme

Function Prototype

Jpg_Decompress([ in ] BSTR *File_Name, [out,retval] long* Result_Image);

Return Value

Result_Image

long *

returned pointer to the uncompressed image structure, a valid Image pointer on success, a NULL pointer on failure

Parameters

Name

Type

Description

File_Name

BSTR

temporary compressed image file name

Description

JPEG2000 is a wavelet-based image compression standard and coding system. It was created by the Joint Photographic Expert Group committee in the year 2000 with the intention of supersending their original discrete cosine transform-based JPEG standard. JPEG2000 can achieve both lossy and loseless image compression.
