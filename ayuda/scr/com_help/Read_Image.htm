Name

Read_Image

loads an Image structure with data from the specified image file

Function Prototype

Read_Image([ in ] BSTR *File_Name, [out, retval] long* Result_Image);

Return Value

Result_Image

long *

Pointer to newly allocated Image structure loaded with data from <File_Name>

Parameters

Name

Type

Description

File_Name

BSTR *

The file name of the image

Description

Loads an Image structure with data from the specified image file <File_Name>. All data members of the structure are initialized to the specifications of the image file read in.
