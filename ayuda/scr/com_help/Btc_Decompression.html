Name

Btc_Decompression

2 level Block Truncation Coding (BTC) scheme

Function Prototype

Btc_Decompression([ in ] BSTR *File_Name, [out,retval] long* Result_Image);

Return Value

Result_Image

long *

returned pointer to the uncompressed image structure, a valid Image pointer on success, a NULL pointer on failure

Parameters

Name

Type

Description

File_Name

BSTR

temporary compressed image file name

Description

 The image is first divided into blocks, and then the digital representation of each pixel in a block is truncated to one bit by thresholding and moment preserving selection of binary levels. The function code was developed from Chapter 5 Section 6 "Block Truncation Compression" (P176 P181) in "Digital Picture Processing" Volume1, second edition, Azriel Rosenfeld and Avinash C. KakYiyan. Academic Press, INC. San Diego, CA. 1982
