Save an image

The ‘Save’ in CVIPtools is in the ‘File->Save’ option on the menu bar. To use it, just select the image by clicking on it and then choose this ‘Save’ option. The image will be saved with the current name in the ‘images’ folder. You can select either one of the following types to save the image from the ‘Save as type’ pop-up menu:

 

Windows Bitmap (*.bmp)

JPEG (*.jpg)

Tagged Image File Format (*.tif)

Compuserve Graphic Interchange (*.gif)

CVIPtools VIP Format (*.vip)

Portable Pixelmap (*.ppm)

Portable Graymap (*.pgm)

Portable Bitmap (*.pbm)

Sun Raster Image (*.ras)

SGI Image File (*.iris)

ITEX Image File (*.itx)

Encapsulated Postscript (*.eps)

Postscript (*.ps)

If you want to save an image in more than one file type, be sure to click on its name in the image queue before each ‘Save’ operation. You can also use the floppy disk symbol just below the menu bar to save the image.
