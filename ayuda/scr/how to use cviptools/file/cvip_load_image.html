Loading an image

If you want to load an image in the image queue, select the ‘File->Open’ option from the menu bar and browse to the folder where the required image is stored. Select the image and press the ‘Open’ button.  The selected will be loaded and displayed. It will also be added to the image queue list.

Currently, CVIPtools will read in the following image file formats:


EPS     (Encapsulated Postscript: monochrome only)
GIF      (Graphics Interchange Format)
IRIS     (Silicon Graphics)
ITX      (Itex)
JPEG   (Joint Photographics Expert Group)
PPM    (Portable PixMap: PNM, PPM, PGM, PBM)
RAS     (Sun Raster files)
TIF      (Tagged Image File Format)
VIP      (Visualization in Image Processing/CVIPtools format)
BMP    (Windows or OS /2 Bitmap)

PS        (Postscript)
