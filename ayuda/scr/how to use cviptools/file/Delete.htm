Delete:

 

This option deletes(closes) the selected image in the image queue from the program. To use, select the image to be deleted and press the ‘Delete’ button just above the image queue.

 

Delete all:

 

This option deletes(closes) all the images in the image queue from the program. Before deleting the images from the program it prompts the user to save the unsaved images. To use, press the ‘Delete all’ button just above the image queue.

 

Lock Input:

 

This option locks the selected image in the image queue, on which the user wishes to perform a number of different operations. Selecting this option allows the user to continue working on the same image instead of going back to selecting the same image repeatedly after performing every operation.
