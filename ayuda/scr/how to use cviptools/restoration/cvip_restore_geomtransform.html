Restoration: Geometric Transforms

Here you can restore an image that has been spatially (geometrically) distorted using the method of quadrilateral tiepoints. You can also perform the opposite, that is, image distortion or image warping.

The restoration or warping takes place by a using a mesh file which contains the (r,c) coordinates pairs for the irregular grid.

Note that to restore an image that has been distorted with this method, simply reverse the direction, select the distorted image, and use the same mesh file.

To turn On/Off the mesh display, right click on the image and select the ‘Draw/Show Mesh’ option by clicking the left mouse button as desired.

In general, to perform the geometric transform: enter a mesh file name, select direction, select gray level interpolation method and press ‘Apply’. There are three methods for using/generating the mesh file, as follows.

    TO CREATE A NEW MESH FILE:

Select ‘Enter a new mesh file’ radio button at top of the window.

Enter or select the number of rows and columns of mesh points.

Now enter the mesh points. For entering mesh points press the ‘Alt +  click Left mouse button’ to select all mesh points one by one. For example: If you select 3 rows and 3 columns, you have to enter 9 mesh points. The number of mesh points left to enter is displayed in a text box. When all the mesh points are entered by the user the program prompts the user to press the ‘Apply’ button and perform the operation.

As the points are being entered, the mesh will be displayed. To turn On/Off the mesh display, right click on the image and select the ‘Draw/Show Mesh’ option appropriately as desired. If the mesh is NOT acceptable, press the ‘Redo’ button and enter a new mesh. You may either keep or change the mesh file name and number of rows and columns of mesh points. You must enter new mesh coordinates.

After you have an acceptable mesh, select the direction of the transformation and the gray value interpolation method. Press ‘Apply’ to apply the mesh to the current image.

To save the mesh you just used, click the ‘Save Mesh’ button. As default, it will be saved in $CVIPtoolsHOME\bin\mesh directory.

    TO USE AN EXISTING MESH FILE:

Select ‘Use an existing mesh file’ radio button at the top of the window.

Press the ‘Select’ button. This will take you to the list of existing mesh files in the ‘mesh’ directory. Select one mesh file from the list of  existing mesh files. 

Select the direction of the transformation and the gray value interpolation method. Press ‘Apply’ to apply the mesh to the current image.

    TO CREATE A SINE MESH

Select ‘Create a sine mesh’ radio button at the top of the window.

Enter a mesh file name. If the file name already exists, you will be prompted to change the name.

Enter or select the number of rows or columns of mesh points. The same number will be used for both rows and columns.

Note: for a 256 x 256 image, create a sine mesh of 16 x 16 or smaller. For a 512 x 512 image, the maximum mesh size will be 32 x 32.

Press the ‘Apply’ button. A mesh will be created and displayed, and will be applied to the current image.

The mesh has its upper-left corner located at column 0, row 25. Its approximate dimensions will be 15*(number of rows) x 15*(number of columns).

To save the Sine mesh you just applied to the image, click the ‘Save Mesh’ button. As default, it will be saved in $CVIPtoolsHOME\bin\mesh directory.
