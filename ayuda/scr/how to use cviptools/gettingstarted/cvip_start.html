Getting Started

Welcome to CVIPtools. This program allows you apply various computer imaging techniques to digital images, and compare results from different algorithms.

To start, you must first load an image. For loading an image, select the ‘Open’ option from the ‘File’ menu and browse to find the image to be loaded. Select the image and press the open button. The selected image is loaded.  Note that starting with version 5.3 blocks of images can be selected and loaded using the Windows standards of the <Shift> and <Ctrl> keys to mark more than one image.

After loading an image, the image(s) will be displayed and the name(s) will appear in the image queue (see below) on the left of the main window.

Next, click the mouse on one of the selections across the top of the main window: Analysis, Restoration, Enhancement, Compression, or Utilties and the corresponding pop-up menu will appear. Select the category of operations desired from the pop-up menu and it will expand into a window that contains all the functions for that category. In general, a specific function is selected by the radiobuttons on the left of the window, and the ‘Apply’ button in the lower right will perform the operation. The ‘Cancel’ button will remove the window from the screen.

See the help pages on the individual functions to learn about them.

    The Image Queue

The image queue, displayed on the left of the main window, is the place to select images for processing. This is done by clicking on the name in the queue with the left mouse button. The image queue can have a long list of images but the current image is highlighted. Any image from the list can be made the current image by clicking on it. The ‘Delete’ button ( Located on the top-left corner just below the toolbar) removes the current image from the image queue. The ‘Delete all’ button removes all the images from the image queue. Note that starting with version 5.3 groups of selected images can be deleted using the Windows standards of the <Shift> and <Ctrl> keys to mark more than one image. The <Delete> key or the button on the GUI can be used to delete the selected images. If the images in the image queue have not been previously saved, a message box will appear just to confirm that the unsaved images will be lost. The ‘Lock Input’ checkbox (when checked) helps to lock onto an image as the current image if the user wants to perform various operations on the same image. The image queue can be made to appear or disappear by using the ‘View->Image Queue’ option from the menu bar. Note that the image queue is RAM-based and contains the images in the form of CVIPtools image structures, which will be remapped for display purposes, if needed.

    Function Parameter Entry

All CVIPtools functions have default parameter values that will provide reasonable and useful results. Additionally, the pop-up menus have lists of values that are typical and/or useful for most parameters. If you are unfamiliar with a function, try the default values first. If you don’t get what you want then try the pop-up selections. However, do not feel limited to these values - most parameters allow entry of any value you like and will not allow entry of values that will bomb the program. Also, in computer imaging applications, the results you get are image and application dependent, so our list of typical values may not work for your particular application. In some cases you may want to get "nonstandard" results. So, live dangerously, explore, enter your own values.

If a filename is too long to fit in a text box after the user selects the file (the path is too long), just place your cursor on the text box after selecting the required file to display its complete filename.

There are many methods used for data entry:

- An entry box with an arrow button to its right allows you to click on the arrow with the mouse and drag the mouse over the desired value on the list. The selected value will appear in the entry box when you release the mouse button.

- With functions that need two images, the second image name can be selected from the pop-up menu and the current image will be the first image. For examples see arithmetic operations under Utilities->Arith/Logic.

See specific help pages for the functions you want to use for help with required parameters.

    Quitting CVIPtools

In the main window, select File->Exit. This will bring up an Alert box, which will ask you if you really want to exit. If you select ‘yes’, be aware that all images in the queue that have not been written to disk will be lost. Note: images are written to disk with a Save operation.
