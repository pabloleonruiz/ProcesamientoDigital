Enhancement: Pseudocolor

Pseudocolor works on grayscale and binary images. Pseudocolor is often used to enhance images since the human visual system can perceive thousands of colors in a small spatial area, but only about 100 gray levels.

For All Functions:
Frequency domain mapping:
Gray level mapping:

Gray level mapping II:
Intensity slicing:

    FOR ALL FUNCTIONS:

After setting the desired parameters, press the ‘Apply’ button at lower right to perform the operation.

The current image is automatically selected, and can be changed by clicking mouse button 1 on an image in the main window’s image queue.

    Frequency domain mapping:

This works by first performing a Fourier transform on the image, filtering, followed by an inverse Fourier transform. The spectral regions used for the filters are circular and ring-shaped, with a low-pass region about the origin specified by the low cut-off, a band-pass region specified by the low and high cut-off, and the high-pass region specified by the high cut-off. Then the inverse Fourier transform is performed on each of the filtered outputs separately, and the results from each of the bands are put into either the red, green or blue band of a color image.

To use, select or enter low and high cut-off frequencies, then select a color for the low-pass, band-pass, and high-pass bands. Each band must be a different color (R, G, or B).
        

    Gray level mapping:

Gray level mapping pseudocolor works by defining a different mapping equation for each of the red, green and blue bands of the output color image. The input is the original gray level values.

To use, select a shape for each band from its pop-up menu. Note that customized gray level mapping/pseudocolor can be done using Enhancement->Histograms->Linear Modification in conjunction with Utilities->Create->Extract and Utilities->Create->Assemble bands.

 

    Gray level mapping II:

Gray level mapping pseudocolor works by defining a different mapping equation for each of the red, green and blue bands of the output color image. The inputs are the original gray level values.

To use, select the required mapping file from the pop-up menu or load it from a different directory or just type in the filename in the text box provided. Select a Gray scale image on which the mapping is to be performed. You can also make your own map by pressing the ‘Custom Remap Curve’ button at the bottom of the ‘Enhancement’ window. Here, you can select either the ‘Linear’ or ‘Lagrange Interpolation’ method. Select the ‘Red’, ‘Blue’ or ‘Green’ band and then make your own curve. The ‘Y’ axis represents the ‘Remapped Values’ and the ‘X’ axis represents the ‘Original Values’. To make a common curve for all bands select ‘All’ and then make the curve. You can make a curve by selecting any point in the ‘Interpolation Box’ by a left mouse click. The left mouse button is also used to drag the points. The points at the ends can only move up and down while the points created at other locations can move anywhere inside the box. To get rid of any point, just get the mouse on that point and click the right mouse button. You can also add the points manually by selecting or entering the ‘X’ and ‘Y’ co-ordinates from the pop-up menu. Press ‘Add’ after entering each point. After you make a curve for all the bands, press ‘All’ to view all the curves together. Then press ‘Save’ to save this map. When this map is saved it appears in the main windows pop-up menu. You can then select it and press ‘Apply’ at the lower right of the ‘Enhancement’ window to perform the gray level mapping.

 

     Intensity Slicing:

Here you can specify up to four ranges of gray levels and the RGB values for the colors to which you want to map each range of gray level values.

 

Intensity slicing will change all pixels with values in the range ‘From’ to the range ‘To’ from their gray values to a color with RGB values selected or entered in the boxes to the right of the range.

In the topmost ‘From’ box, select or enter a value, then select or enter a larger value in the ‘To’ box beside it. Then select or enter the RGB values in the boxes in the same row as the desired range.

You may designate up to four ranges for slicing. Use the rows in order from top to bottom, and each new row must start ‘From’ a larger value than the preceding row’s ‘To’ value.

Checking the ‘Set Out of Range Data to 0’ box will cause the selected range(s) to be returned on a black image; not selecting or deselecting it will return the original image with color highlights in the selected range(s).
