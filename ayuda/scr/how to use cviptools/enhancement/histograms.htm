Enhancement: Histogram/Contrast

These functions modify the distribution of gray levels in an image, thereby changing the brightness and contrast. They do this by controlling the shape or distribution of the output image histogram. In addition to the histogram modification functions, the gray level mapping functions are included here, since they also have the end result of modifying the histogram.

    FOR ALL FUNCTIONS:

After setting the desired parameters, press the ‘Apply’ button at lower right to perform the operation.

The current image is automatically selected, and can be changed by clicking mouse button 1 on an image in the main window’s image queue

    Histogram Equalization:

Histogram equalization is a standard method to improve the contrast in an image. If the original image has very poor contrast, you will see dramatic results.

To use the function:

For gray-scale images, no parameters are needed.

For color images, select the band to use as a basis for equalization from ‘On Band’ buttons. Selection of ‘Value’ will result in the image being transformed to the HSL colorspace, equalization of the brightness (‘Lightness’), and inverse transform to an RGB image.

Note that histogram equalization can be done by selecting the image as current image and pressing ‘e’ on the keyboard. However, when this is done, the equalized image is represented with the same name as the current image (overwrites the current image), in the image queue.

    Histogram Slide:

Sliding the histogram up will brighten the image and sliding it down will darken the image.

To use, select slide direction, and enter or select from pop-up menu the amount of slide.

    Histogram Shrink:

This will reduce the contrast in an image. It may be a useful preprocessing step in a compression algorithm (followed by a stretch after decompression).

To use, enter or select from pop-up menus the lower and upper values.

    Histogram Stretch:

This will improve the contrast in an image. By clipping a small percentage at the ends, you can avoid the possibility of a few high or low pixel values compromising the stretch effect.

To use, enter or select from pop-up menus the low and high limit values, and the percent to clip from low and high ends.

    Local Equalization:

This performs histogram equalization on small blocks throughout the image. This will let you see more local detail, often in areas that may appear constant.

To use:

For gray-scale images, set the desired blocksize.

For color images, select the band to use as a basis for equalization from ‘On Band’ buttons, then set the desired blocksize. A black border will appear on the output image as a result of the selected blocksize.

    Color Contrast Algorithm:

This will improve the color of the image. To use, select a color image and by entering or selecting the low and high limit values, and the percent to clip from low and high ends values from pop-up menus and hit apply button. The improvement of the color of the image depends on the values chosen for the low limit, high limit, low clip and high clip.

    Histogram Specification:

You will specify the histogram modification equation for each band in the image separately. The equation can include algebraic operators (+,-,*,/,^), and any of the following functions:

sin(arg)             sinh(arg)           imp(arg)           sqrt(arg)
cos(arg)           cosh(arg)          floor(arg)          abs(arg)
tan(arg)            tanh(arg)           step(arg)           ceil(arg)
asin(arg)           exp(arg)           u(arg)               deg(arg)
acos(arg)          ramp(arg)         rad(arg)        
atan(arg)           r(arg)                log(arg)         

For gray-scale images, type the desired equation into the ‘Formula for R band’ equation box or select an equation from the pop-up menu to the right of the box. The following equations are included in the pop-up menu as examples:

sin(0.025*x)      ramp(2.0*x+5)
x+25                  exp(0.025*x)
x*2.0                 log(0.5*x+2)

For example, to specify a tangent, enter: tan(100*x); to use sqrt, you may enter: sqrt(x+10).

For color images, type equations for each band or select equations from pop-up menus.

    Linear Modification:

Enter integers for the starting and ending gray values to change and a new initial gray level. Then enter a number for the slope of the line to be used to modify the gray levels within the specified range. The slope may be positive, negative, or zero. A zero slope will cause all values in the given range to be mapped to the ‘Initial value’ gray level. Finally, you may either retain values outside the specified range or set them to zero.

This function can be used in conjunction with Utilities->Create->Extract bands and Utilities-> Create->Assemble bands to customize a gray level mapping pseudocolor image.

    Gray Level Multiplication:

Select with or without ‘Byte clipping’, then enter the multiplication factor or select it from the pop-up menu. If you choose with ‘Byte clipping’, the image is first remapped to Byte, then is multiplied by the factor and truncated to Byte. If you choose without ‘Byte clipping’, data is converted to float, multiplied by the factor, and a float image is put into the image queue. The returned image is remapped to Byte for display only.

Note that another multiplication option is available with Utilities->Arith/Logic->Multiply. There you can multiply two images.
