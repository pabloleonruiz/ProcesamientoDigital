    FOR ALL FUNCTIONS:

After setting the desired parameters, press the ‘Apply’ button at lower right to perform the operation.

For all of the following functions, the current image is automatically selected, and can be changed by clicking mouse button 1 on an image in the main window’s image queue.

    Crop:

The crop function allows you cut a subimage out of the current image and put it in the IMAGE QUEUE. This may be done to select a region of interest (ROI), or to paste a part of one image into another with the following ‘Paste’ function.

To crop an image, the width, height and upper left coordinates of the subimage can be entered directly in the entry boxes or selected from the pop-up menu.

Alternately, you can select the crop area by dragging with mouse button 1 on the current image with the ‘Shift’ key pressed. If you don’t like the size of the rectangle, you can make a new one on top of the present one. When you are done making a new rectangle the previous one gets deleted automatically. Note that, although the coordinates are used automatically when you press ‘Apply’, they will not appear in the window until AFTER the ‘Apply’ button has been pressed.

Once, the parameters are set as desired, press ‘Apply’ in the lower right corner.

    Paste:

This function allows you to paste an image, or subimage, onto another image. The standard method will replace the underlying pixel values in the original image with the pasted image. Alternately, this can be done transparently, which adds the two together so that the original image is still visible under the pasted image.

To select an image to be pasted into the current image you can select the name from the pop-up menu. Next, type in the row and column coordinate of the upper left corner of location in the current image where you desire to paste the image or select the row and column coordinates from the pop-up menu. Finally, a transparent paste can be selected, if desired.

Once, the parameters are set as desired, press ‘Apply’ in the lower right corner.

    Resize:

With the resize function you can enlarge or shrink an image. Additionally, you can spatially distort an image by enlarging it in one dimension and shrinking it in the other dimension. The desired new dimensions in height and width are entered into the entry boxes or selected from the pop-up menu. Once, the parameters are set as desired, press ‘Apply’ in the lower right corner.

    Rotate:

This function will rotate an image and place the result within a black rectangle large enough to contain the entire rotated image.

To perform a rotation, enter or select a float or integer value for the number of degrees of rotation. A positive number will rotate the image clockwise, and a negative number will rotate counterclockwise. Once, the parameters are set as desired, press ‘Apply’ in the lower right corner.

Note that this function will handle complex images, for example, an fft spectrum.

    Translate:

This function will shift an image left or right, and up or down. It allows for wrapping the shifted areas onto the opposite side of the image or to let the portions shifted out be filled with specific gray level values.

To perform a translation, select the directions and enter the desired amounts. Then select the method of dealing with parts of the image uncovered by the translation. ‘Wrap the image...’ will cause those parts of the image that move beyond an edge to reappear at the opposite edge; the entire image will still be present, but will be rearranged. ‘Fill the area...’ will cause areas left behind by moving the image to be filled with a uniform gray value you enter. Once, the parameters are set as desired, press ‘Apply’ in the lower right corner.

    Zoom:

This function performs a zoom operation and can also be used to enlarge an image by zooming the entire image. Two methods are available for the zooming: zero-order or linear interpolation. In general, the linear interpolation method will provide more visually pleasing results, but will take longer to perform. The zero-order method will result in blocky effects, but is faster and does not “create data” - the linear interpolation method “creates data” via interpolation, which is undesirable in some applications.

To zoom in on an image quadrant, choose the quadrant of the image to expand. Then enter a factor by which to expand the quadrant. If you want to zoom an area other than a fixed quadrant, first select ‘the area beginning from...’ button. You then choose the area to zoom by entering or selecting the upper left row and column coordinates and the height and width values into the entry boxes.

Alternately, you can select the area to zoom by dragging with mouse button 1 on the current image while keeping the ‘Shift’ key pressed. If you don’t like the size of the rectangle you can redraw a new one on the first one. Note that, although the coordinates are used automatically when you press ‘Apply’, they will not appear in the window until AFTER the ‘Apply’ button has been pressed.

Next, enter the factor by which you want to expand the image in the window labeled ‘By’. Next, select ‘zero order’ or ‘linear interpolation’ for the method in the window labeled ‘pixels’. Once, the parameters are set as desired, press ‘Apply’ in the lower right corner.
