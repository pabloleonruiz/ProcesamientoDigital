NAME

    ObjectManager  -   CVIPtools   toolkit   library   for   object
     management/handling
 

SYNOPSIS

      #include <CVIProi.h>
      #include <DLL.h>
      #include <LL.h>
      #include <Link.h>
      #include <HT.h>
      #include <Stack.h>
 

PATH

    $CVIPtoolsHOME\lib
 

DESCRIPTION

     ObjectManager is the home of the  linked  list,  double  linked
     list, hash table, and stack handling functions.
 

TYPES AND CONSTANTS

     See individual header files in $CVIPtoolsHOME\include.
 

RETURN VALUES

     See individual header files in $CVIPtoolsHOME\include.
 

SEE ALSO

     DLL.c, HT.c, LL.c, Stack.c
