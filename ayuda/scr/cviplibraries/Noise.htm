NAME

     Noise- CVIPtools core toolbox library for noise  genera-
     tion
 

SYNOPSIS

      #include <CVIPtoolkit.h>
      #include <CVIPdef.h>
      #include <CVIPimage.h>
      #include <CVIPconvert.h>
      #include <CVIPnoise.h>
 
     See the individual function man pages for details and  exam-
     ples.
 

PATH

        $CVIPtoolsHOME\lib
     
 

TYPES AND CONSTANTS

      #define FS 8000
      #define NOISE 100*cos(2*Pi*T*1000/FS)
      #define trunc(x) (((x) > 255) ? 255:(((x) < 0) ? 0:(x)))
 

FUNCTIONS

     The following functions are included in Noise:
 
                            gamma_noise
                            gaussian_noise
                            neg_exp_noise
                            rayleigh_noise
                            speckle_noise
                            uniform_noise
     
     The C files containing these functions are located under $CVIPtoolsHOME\CVIPC\NOISE
 

RETURN VALUES

     The functions return a modified Image  pointer  on  success,
     and returns a NULL pointer on failure.
