NAME

Feature - CVIPtools  core  toolbox  library  for  feature
          extraction and pattern classification routines
 

SYNOPSIS

      #include < CVIPtoolkit.h>
      #include < CVIPdef.h>
      #include < CVIPimage.h>
      #include < CVIPconvert.h>
      #include < CVIPfeatures.h>
 
     See the individual function man page for details  and  exam-
     ples.
 

PATH

     $CVIPtoolsHOME\lib
