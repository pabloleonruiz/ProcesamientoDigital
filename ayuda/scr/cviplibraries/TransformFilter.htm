NAME

    TransformFilter - CVIPtools core  toolbox  library  for  two-
     dimensional transform-domain filtering
 

SYNOPSIS

      #include <CVIPtoolkit.h>
      #include <CVIPdef.h>
      #include <CVIPimage.h>
      #include <CVIPconvert.h>
      #include <CVIPtransform.h>
      #include <CVIPxformfilter.h>
 
     See the individual function man pages for details and  exam-
     ples.
 

PATH

        $CVIPtoolsHOME\lib
 

FUNCTIONS

     The following functions are included in TransformFilter:
 
      Butterworth_Band_Pass
      Butterworth_Band_Reject
      Butterworth_High
      Butterworth_Low
      geometric_mean
      High_Freq_Emphasis
      h_image
      homomorphic
      Ideal_Band_Pass
      Ideal_Band_Reject
      Ideal_High
      Ideal_Low
      inverse_xformfilter
      least_squares
      nonfft_xformfilter
      notch
      parametric_wiener
      power_spect_eq
      simple_wiener
      wiener
 
     The C files containing these functions are located under $CVIPtoolsHOME\CVIPC\XFORMFILTER
 
 

RETURN VALUES

     Image pointer to filtered transform data
 

SEE ALSO

     Specific man pages for functions listed above
