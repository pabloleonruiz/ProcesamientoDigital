NAME

    Color - CVIPtools core toolbox  library  containing  rou-
     tines related to color transforms
 

SYNOPSIS

      #include <CVIPtoolkit.h>
      #include <CVIPdef.h>
      #include <CVIPimage.h>
      #include <CVIPconvert.h>
      #include <CVIPcolor.h>
 
     See the individual function man page for details and examples.
 

FUNCTIONS

     Color includes the following utilities:
 
             colorxform
             color_contrast_enhance
             ipct
             linxform
             luminance_Image
             lum_average
             pct
             pct_color
             pseudocol_freq
 
     The C files containing these functions are located under $CVIPtoolsHOME\CVIPC\COLOR
 

DESCRIPTION

     see individual manual pages for each function
