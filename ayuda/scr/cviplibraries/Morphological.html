NAME

    Morphological - CVIPtools core toolbox library for morphology
 

SYNOPSIS

      #include <CVIPtoolkit.h>
      #include <CVIPdef.h>
      #include <CVIPimage.h>
      #include <CVIPconvert.h>
      #include <CVIPmorph.h>
 
     See the individual function man pages for details and  exam-
     ples.
 

PATH

   $CVIPtoolsHOME\lib
 

FUNCTIONS

     The following functions are included in Morphological:
 
      hexrotate
      MorphClose
      MorphClose_Image
      MorphDilate
      MorphDilate_Image
      MorphErode
      MorphErode_Image
      morpho
      morphIterMod_Image
      MorphOpen
      MorphOpen_Image
      Morph_Hitmiss
      Morph_Thinning
      Morph_Skeleton
 
    The C files containing these functions are located under $CVIPtoolsHOME\CVIPC\MORPOLOGICAL
 

RETURN VALUES

     See the individual function man pages
 

SEE ALSO

     Functions listed above
