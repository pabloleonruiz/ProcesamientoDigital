NAME

     Geometry - CVIPtools core toolbox library  for  geometric
     manipulations
 

SYNOPSIS

      #include <CVIPtoolkit.h>
      #include <CVIPdef.h>
      #include <CVIPimage.h>
      #include <CVIPconvert.h>
      #include <CVIPgeometry.h>
 
     See the individual function man pages for details and  exam-
     ples.
 

PATH

        $CVIPtoolsHOME\lib

     The following functions are included in Geometry:
 
                          bilinear_interp
                          copy_paste
                          create_black
                          creat_checkboard

                          create_circle
                          create_degenerate_circle
                          create_ellipse
                          create_line
                          create_rectangle
                          create_sine
                          create_squarewave
                          crop
                          display_mesh
                          enlarge
                          keyboard_to_mesh
                          mesh_warping
                          mesh_to_file
                          object_crop
                          rotate
                          shrink
                          solve_c
                          spatial_quant
                          translate
                          zoom
 
      The C files containing these functions are located under $CVIPtoolsHOME\CVIPC\GEOMETRY
